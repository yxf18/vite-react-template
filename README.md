## 技术栈
React、Route Router19、Ant Design、Vite、TypeScript、
Mobx、@emotion/react、Axios

## 规范化
eslint、prettier、commintlint

如何绕过commitlint(不推荐使用，除非紧急情况需要绕过校验规则)
```bash
git commit -m "xxx" --no-verify
```

## 项目脚本
```bash
yarn // 安装依赖
npm run dev // 启动项目
npm run build // 打包项目
npm run lint // 代码检查
```


> <font color="red">node版本：20.x</font>
